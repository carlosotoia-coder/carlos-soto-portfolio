## Proyectos destacados

### Red Social para Personas Neurodivergentes

Plataforma social diseñada para facilitar la amistad y las relaciones entre personas neurodivergentes en un entorno seguro y accesible.

Objetivo del proyecto:
Crear un espacio digital donde las personas puedan conectar sin las barreras sociales tradicionales.

Tecnologías previstas:
- Django
- PostgreSQL
- HTML
- CSS
- JavaScript

---

### Ecosistema de Aprendizaje para Niños con TEA

Aplicación educativa que ayuda a niños con autismo a aprender habilidades de la vida diaria mediante simulaciones de situaciones cotidianas.

Objetivo del proyecto:
Facilitar el aprendizaje de rutinas y contextos sociales mediante tecnología interactiva.

Tecnologías previstas:
- Python
- Django
- JavaScript
- Base de datos PostgreSQL

---

### Academia Deportiva de Fútbol

Aplicación web para gestionar una academia deportiva para adultos.

Funciones del sistema:
- Registro de jugadores
- Planificación de entrenamientos
- Seguimiento del rendimiento deportivo

Tecnologías previstas:
- Django
- PostgreSQL
- HTML
- CSS
- JavaScript
